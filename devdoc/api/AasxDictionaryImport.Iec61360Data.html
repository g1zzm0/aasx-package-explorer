    Class Iec61360Data

    The data for an element according to the IEC 61360 standard. This data can be used to generate AAS elements, especially submodels, properties and collections. The idShort for the generated AAS element is derived from the short name or the preferred name.

    It is not necessary to set all fields -- some are only applicable for some element types. But every data object must have a unique IRDI.
