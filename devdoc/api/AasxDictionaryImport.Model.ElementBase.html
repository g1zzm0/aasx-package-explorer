    ImportSubmodelElementsInto(AdminShellV20.AdministrationShellEnv, AdminShellV20.IManageSubmodelElements)

    Converts this element into a AAS submodel element (i. e. a property or a collection) and adds it to the given parent element (typically a submodel). If this element cannot be converted into a submodel element, this method returns false.

    Implementations of this method should check the IsSelected attribute both for this element and for all children. Only those elements with IsSelected equals true should be imported.
